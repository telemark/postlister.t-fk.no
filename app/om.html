Om offentlig journal

Postlister legges ut kl 14 to arbeidsdager etter at de er journalført.
Postlistene for en gitt dag er tilgjengelig i 90 kalenderdager.

Om løsningen

Dette er en egenutviklet løsning fra Telemark fylkeskommune.
Den består av tre deler som alle er fritt tilgjengelig fra GitHub

  1. si-import-openjournal
    Script som importerer journalpostene fra Public 360 til MongoDB
  2. api.t-fk.no
    API serveren som publiserer alle de åpne webservicene våre på https://api.t-fk.no/
    Basert på MongoDB, io.js og hapi
  3. postlister.t-fk.no
    Nettstedet du er på nå. Basert på Web Starter Kit, React og Browserify.
